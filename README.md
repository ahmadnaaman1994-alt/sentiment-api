# Sentiment Analysis API

Короткий API для анализа тональности текста, созданный с использованием FastAPI и модели HuggingFace.

## Эндпоинты
GET /

Проверка работы API.

## POST /predict

Принимает текст и возвращает:

оценку тональности (1–5)

уровень уверенности модели

## Пример запуска
```
pip install -r requirements.txt
uvicorn main:app --reload
```
Документация после запуска:
http://127.0.0.1:8000/docs
